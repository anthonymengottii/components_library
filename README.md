## ✨ Por que Components Library?

A Components Library ajuda você a **entregar interfaces incríveis mais rápido**. Em vez de gastar horas criando animações do zero, escolha um componente polido e personalize-o para combinar com sua visão.

> 💬 **Animações de Texto** · 🌀 **Animações** · 🧩 **Componentes** · 🖼️ **Fundos**


## 🎯 Direção do Projeto

Este projeto começou tendo o **React Bits** como base e referência de padrões de componentes e estrutura de documentação.

Hoje, o objetivo evoluiu: a Components Library está indo além de componentes individuais para oferecer **templates completos, experiências iniciais e fluxos mais amplos de construção de UI**.

## 🙏 Créditos

Agradecimento especial ao **[React Bits](https://www.reactbits.dev/)** pela inspiração que ajudou a dar início a esta biblioteca.

## 🚀 Recursos

- **110+ componentes** — animações de texto, elementos de UI e fundos, com crescimento semanal
- **Dependências mínimas** — leve e com suporte a tree-shaking
- **Totalmente customizável** — ajuste tudo via props ou edite o código-fonte diretamente
- **4 variantes por componente** — JS-CSS, JS-TW, TS-CSS, TS-TW (para todos os gostos)
- **Pronto para copiar e colar** — funciona com qualquer projeto React moderno

## 📦 Instalação

A Components Library oferece suporte ao [shadcn](https://ui.shadcn.com/) e ao [jsrepo](https://jsrepo.dev) para instalação rápida via CLI.

```bash
# Exemplo: adicionar um componente via shadcn
npx shadcn@latest add @components-library/BlurText-TS-TW
```

Cada página de componente inclui comandos de CLI prontos para copiar. Veja o [guia de instalação](https://github.com/anthonymengottii/components_library/get-started/installation) para mais detalhes.

Você também pode selecionar suas tecnologias preferidas e copiar o código manualmente.
